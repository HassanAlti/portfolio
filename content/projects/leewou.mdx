Leewou is an influencer marketing platform designed to help brands discover,
analyze, and track campaigns with over 350 million influencers across TikTok,
Instagram, and YouTube. Built with modern web technologies and powered by
Elasticsearch for lightning-fast search across massive datasets.

**Status:** Early Beta (Development started January 2026)

## Overview

Leewou simplifies influencer marketing by providing brands with powerful search
capabilities, deep analytics, and campaign tracking tools. Whether you're
looking for micro-influencers or tracking enterprise-level campaigns, Leewou
delivers insights at scale.

## Key Features

- **Massive Database**: Search through 350M+ influencers across major platforms
- **Deep Analytics**: Comprehensive metrics including engagement rates, audience
  demographics, and growth trends
- **Campaign Tracking**: Monitor and manage multiple influencer campaigns in one
  place
- **Advanced Filters**: Filter by platform, niche, follower count, engagement
  rate, and more
- **Lightning-Fast Search**: Elasticsearch-powered search returns results in
  milliseconds
- **Real-time Monitoring**: Track influencer performance and content updates
- **Secure Access**: JWT-based authentication with role management

## Technologies

### Frontend

- **Next.js**: React framework for optimal performance and SEO
- **Shadcn/ui**: Modern, accessible component library
- **Tailwind CSS**: Utility-first styling for responsive design
- **TypeScript**: Type-safe development

### Backend

- **Express.js**: Fast and minimalist web framework
- **Elasticsearch**: Distributed search engine for massive-scale data
- **JWT Authentication**: Secure token-based authentication
- **Node.js**: JavaScript runtime for scalable backend
